ABOUT BAZINGA:
->BAZINGA IS AN AI INTEGRATED MODEL THAT ENABLES USER TO EASE THERE WORKLOAD BY DOING MOST OF THE TASK.
->BAZINGA IS AN AI-POWERED ENGINEERING AGENT THAT DESIGNS, DRAFTS AND EXECUTES REAL WORLD ACTIONS(CODE, GITHUB,BLOGS,LINKEDIN) WITH HUMAN APPROVAL.


WORK_FLOW:
  I. USER ENTERS THERE TASK THAT IS TO BE EXECUTED. FOR EXAMPLE MAKE A PROJECT THAT SOLVES MATHEMATICAL NOTATIONS AND UPLOAD IT TO MY GIT.
  II. BAZINGA WILL TAKE THE INPUT AND GENERATE THE CODE , WORKFLOW, DIAGRAM,ETC.
  III.BEFORE ADDING THE PROJECT TO GIT, IT WILL ASK THE USER TO VERIFY IF THE PROJECT CODE,DIAGRAMS,WORKING,ETC ARE IS UPTO THERE SATISFACTION.
  IV. WHEN GIVEN PERMISSION,BAZINGA UPLOADS THE APPROVED PROJECT TO GITHUB. 

FEATURES:
 1. CAN CREATE PROJECT ON ANY LANGUAGE.
 2. MAKE MANAGEMENT OF PROJECTS EASY FOR THE USER.
 3. AI INTEGRATED PLATFORM THAT NEEDS HUMAN APPROVAL.


SUCCESS CRITERIA:
BAZINGA_V1 IS CONSIDERED SUCCESSFUL IF:
1. A GITHUB REPOSITORY IS CREATED AUTOMATICALLY 
2. GENERATED CODE RUNS WITHOUT ERRORS 
3. README CLEARLY EXPLAINS THE PROJECT 
4. NO GITHUB ACTION OCCURS WITHOUT USER APPROVAL
